# fmt

The `fmt` module provides functions for formatting and printing strings.

<Callout type="warning">
This module is deprecated. The top-level `print`, `printf`, and `sprintf`
builtins should be used instead of the functions in this module. See the
documentation for the top-level built-ins [here](/docs/builtins#print).
</Callout>
